# Edible Food Waste Credit Smart Contracts

This repository contains the smart contract implementations developed for the Edible Food Waste Credit project.

**"Edible Food Waste Credit"**

## Contents

- **EFWCreditFT.sol**  
  Smart contract for fungible token issuance and management within the Edible Food Waste Credit system.

- **EFWRedemption.sol**  
  Smart contract for redemption workflow and NFT-related certificate or retirement logic within the Edible Food Waste Credit system.

## Description

This project demonstrates how smart contracts can support traceable and verifiable edible food waste management through tokenization and redemption mechanisms. The contracts are designed to represent key on-chain components of the Edible Food Waste Credit framework, supporting transparent issuance, validation, and redemption processes.
